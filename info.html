this website does not collect any info except that collected by Cloudflare when you visit this site.

in regards to accessability

this website is not currently compliant with WCAG 2.1, by W3C. according to accessMonitor, a tool by acessibilidade.gov.pt, this website scores a 6.0 out of 10.

we are not required to follow accessability rules by law, but there's a soon to be launched version of this website with an accessability widget

in regards on hosting

this website is hosted on OVHCloud's bare metal infrastructure. the machine is running coolify, and this website is running behind a docker container.

all images however are either hosted on catbox.moe or github's cdn.
